Welcome to developer documentation for Leapp!
=============================================

 - NVR schema used for for leapp-related `rpm packages <https://github.com/oamg/leapp-guidelines/blob/master/build-schema.md>`_.
 - Architecture overview of `leapp and snactor tools <https://github.com/leapp-to/leapp-to.github.io/blob/master/architecture.md>`_.
